# ALL-INSTALLER
An open source all in one tool installer using (whiptail).

This tool will install some of my hacking tools on my Github repository.
#
# INSTALLATION & USAGE

    $ git clone https://github.com/mkdirlove/ALL-INSTALLER
    $ cd ALL-INSTALLER
    $ chmod +x *
    $ bash install.sh
    $ bash all-installer.sh
